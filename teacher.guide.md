# 📚 Гайд преподавателя — бот-помощник репетитора

Бот работает в Telegram, а часть действий удобнее делать в **Mini App** (открывается кнопкой меню
в боте). Почти всё, что есть в боте, продублировано в приложении — выбирай, как удобнее.

## 1. Первый запуск

1. Открой бота → **«Старт»** (`/start`).
2. Выбери роль **«📚 Преподаватель»**.
3. Укажи часовой пояс и своё имя (его будут видеть ученики).
4. Бот выдаст **твой код для учеников** — по нему они прикрепляются. Посмотреть код в любой момент:
   **`/mycode`**.

> Совет: сразу создай хотя бы один курс — **`/newcourse`** (например, «Физика, 9 класс»). Тогда при
> подтверждении ученика сможешь сразу назначить ему курс, а в курсе вести план тем.

## 2. Добавление учеников

1. Отправь ученику ссылку на бота и свой код (в личке, не публично).
2. Когда ученик введёт код, тебе придёт сообщение **«К тебе просится ученик: <имя>. Возьмёшь под
   крыло?»** с кнопками ✅ / ❌.
3. Жмёшь ✅ → выбираешь курс (или «Без курса»). Готово — ученик активен.

Заявки также видны в Mini App на вкладке **«Запросы»** (там же подтверждаются переносы и запросы
родителей). **Родители** добавляются по коду самого ученика (его `/mycode`) и подтверждаются
учеником — тебе ничего делать не нужно.

## 3. Расписание

**Регулярные занятия (серия):**
- В боте: **`/newseries`** → ученик → день недели → время → длительность → **на сколько занятий
  вперёд** (4/8/12/16 или «Бессрочно») → ссылка.
- В Mini App: «Уроки» → **«🔁 Регулярные занятия»** — та же форма.

«Бессрочно» = занятия добавляются автоматически на 4 недели вперёд и продлеваются сами. Конечное
число (например, 8) = создаётся ровно 8 занятий до конкретной даты.

**Разовое занятие:**
- В боте: **`/newlesson`**.
- В Mini App: «Уроки» → **«➕ Новое занятие»**.

## 4. Напоминания

**`/reminders`** → можно добавить напоминания:
- **всем ученикам** или **конкретному** ученику;
- за сколько минут до занятия (пресет или своё число).

Важно: твои напоминания **обязательные** — ученик их не отключит. Ученики/родители могут добавлять
только свои личные поверх твоих.

## 5. Материалы и задачи

- **`/material`** — прикрепить PDF (задание) к занятию и задать, за сколько времени до урока бот
  пришлёт его ученику.
- **`/notes`** — прикрепить **конспект** к недавнему занятию: уйдёт **только ученику** после урока.
- **`/gentask`** (или Mini App → **«🧩 Сгенерировать задачи»**) — собрать подборку задач по
  предмету/теме/сложности:
  - метод **«📐 Шаблон»** — задачи с гарантированно верным ответом, без ИИ;
  - метод **«🤖 ИИ»** — через нейросеть (если подключён ключ).
  - На выходе — PDF с **условием** и **решением**. Можно оставить **«Только мне»** или сразу
    отправить выбранному ученику (ему уйдут и условие, и решение).

## 6. После занятия — итог

В Mini App у прошедших занятий появляется раздел **«Требуют отметки»**. Открой занятие → поставь:
- пройденную **тему** (отметится ✅ в плане курса),
- оценку **за работу на уроке** (1–5),
- оценку **за понимание темы** (1–5),
- комментарий.

Итог автоматически уходит **ученику** в чат (родителю оценки не видны).

## 7. Перенос и отмена

**`/reschedule`** (или карточка занятия в Mini App) → выбери занятие → «Перенести»/«Отменить».
Запрос **подтверждает другая сторона** (ученик), так что без согласия время не изменится. Тебе так
же приходят запросы от учеников — подтверждаешь кнопкой.

## 8. План тем

**`/topics`** → ведёшь список тем по курсу. Нажатие на тему меняет статус: ⬜ запланирована →
🔄 в процессе → ✅ пройдена. В Mini App («Темы») можно и создать курс, и добавлять темы.

## 9. Календарь (iCal)

**`/calendar`** даёт ссылку для подписки. Добавь её как «подписку по URL» в Google/Apple Календарь —
все твои занятия (с именем ученика и курсом в названии) будут обновляться сами.

## 10. Прочее

- **`/settings`** — сменить таймзону.
- **`/help`** — список команд.
- Удалить аккаунт со всеми связями можно в Mini App → «Профиль» → «Опасная зона».

## 🧭 Шпаргалка команд

| Команда | Что делает |
|---|---|
| `/start`, `/help` | старт и справка |
| `/mycode` | мой код для учеников |
| `/newcourse` | создать курс |
| `/newseries` | регулярное расписание |
| `/newlesson` | разовое занятие |
| `/reminders` | напоминания (обязательные для учеников) |
| `/material` | прикрепить задание (PDF) |
| `/notes` | конспект ученику после урока |
| `/gentask` | сгенерировать задачи |
| `/topics` | план тем |
| `/reschedule` | перенос/отмена занятия |
| `/calendar` | подписка на расписание (iCal) |
| `/schedule` | ближайшие занятия |
| `/settings` | таймзона |
